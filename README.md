## Metaverse Slayer
Simple card clicker etherium based game 

### What can you do
1. mint your nft character
2. select your nft character
3. kill the boss
4. pick da money
5. repeat

#### Technology used
1) etherium (erc720) + Solidity
2) ganache+truffle
3) React
4) TypeScript
5) sflabs-ui
6) webpack+create-react-app
